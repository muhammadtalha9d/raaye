# Raaye - LinkedIn Content Engine

A modern, responsive website for the Raaye LinkedIn content generation platform.

## 📝 What Changed

The original WordPress PHP file has been converted to:
- **index.html** - Standalone HTML/React application (no PHP required)
- **server.js** - Local Node.js development server
- **package.json** - Project metadata and dependencies

## 🚀 Quick Start

### Option 1: Using Node.js Server (Recommended)

1. Make sure Node.js is installed on your system
2. Open terminal in this directory
3. Run the server:
```bash
node server.js
```
4. Open your browser and navigate to: **http://localhost:3000**

### Option 2: Direct File Opening

Simply double-click `index.html` in Windows Explorer, or open it directly in any modern web browser.
